# كوكيز على الويب - تطبيق إدارة الإعدادات

## نظرة عامة

تطبيق ويب احترافي يسمح للمستخدمين بإنشاء حسابات وتسجيل الدخول وحفظ تفضيلاتهم الشخصية (الاسم، اللغة، لون الخلفية). يستخدم التطبيق Firebase للمصادقة وتخزين البيانات.

## الميزات

- **التسجيل والدخول**: إنشاء حسابات جديدة وتسجيل دخول آمن باستخدام Firebase Authentication
- **إدارة الإعدادات**: حفظ وتحديث تفضيلات المستخدم (الاسم، اللغة، لون الخلفية)
- **تخزين البيانات**: حفظ البيانات في Firestore مع ربط آمن بحساب المستخدم
- **واجهة عربية**: واجهة مستخدم كاملة باللغة العربية
- **تصميم احترافي**: واجهة حديثة وسهلة الاستخدام

## التثبيت والإعداد

### 1. إنشاء مشروع Firebase

1. اذهب إلى [Firebase Console](https://console.firebase.google.com/)
2. انقر على "إنشاء مشروع" وأدخل اسم المشروع
3. اتبع خطوات الإنشاء

### 2. تفعيل المصادقة

1. في Firebase Console، انتقل إلى **Authentication** (المصادقة)
2. انقر على **Get Started** (ابدأ)
3. اختر **Email/Password** (البريد الإلكتروني/كلمة المرور)
4. فعّل هذه الطريقة

### 3. إنشاء قاعدة بيانات Firestore

1. في Firebase Console، انتقل إلى **Firestore Database**
2. انقر على **Create Database** (إنشاء قاعدة بيانات)
3. اختر **Start in test mode** (ابدأ في وضع الاختبار)
4. اختر الموقع الجغرافي المناسب

### 4. الحصول على بيانات المشروع

1. في Firebase Console، انقر على أيقونة الإعدادات ⚙️
2. اختر **Project Settings** (إعدادات المشروع)
3. انسخ بيانات المشروع من قسم **Your apps** (تطبيقاتك)

### 5. تحديث بيانات Firebase

في ملف `script.js`، استبدل `firebaseConfig` بالبيانات الخاصة بك:

```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

### 6. تشغيل التطبيق

1. افتح ملف `index.html` في متصفحك
2. أو استخدم خادم محلي:
   ```bash
   python -m http.server 8000
   ```
   ثم افتح `http://localhost:8000`

## كيفية الاستخدام

### التسجيل

1. أدخل بريدك الإلكتروني وكلمة المرور
2. انقر على زر "تسجيل"
3. سيتم إنشاء حسابك تلقائياً

### تسجيل الدخول

1. أدخل بريدك الإلكتروني وكلمة المرور
2. انقر على زر "دخول"
3. سيتم تحميل إعداداتك السابقة

### حفظ الإعدادات

1. أدخل اسمك
2. اختر اللغة المفضلة
3. اختر لون الخلفية
4. انقر على "حفظ الإعدادات"

### تسجيل الخروج

1. انقر على زر "خروج"
2. سيتم حذف جميع البيانات المحلية

## قواعد أمان Firestore

للسماح بالوصول الآمن، استخدم هذه القواعد في Firestore:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth.uid == userId;
    }
  }
}
```

### بنية البيانات

كل مستخدم يحتوي على الحقول التالية في Firestore:

```javascript
{
  username: string,        // اسم المستخدم
  language: string,        // اللغة المختارة
  bgcolor: string,         // لون الخلفية (بصيغة hex)
  email: string,           // البريد الإلكتروني
  createdAt: timestamp,    // تاريخ الإنشاء
  lastVisit: timestamp     // آخر زيارة
}
```

## استكشاف الأخطاء

### المشكلة: "خطأ في المصادقة"
- تأكد من تفعيل المصادقة في Firebase
- تحقق من صحة بيانات المشروع في `script.js`

### المشكلة: "لا يمكن حفظ البيانات"
- تأكد من إنشاء Firestore Database
- تحقق من قواعد الأمان في Firestore

### المشكلة: "لا يتم تحميل البيانات"
- افتح أدوات المطور (F12) وتحقق من الأخطاء
- تأكد من الاتصال بالإنترنت
